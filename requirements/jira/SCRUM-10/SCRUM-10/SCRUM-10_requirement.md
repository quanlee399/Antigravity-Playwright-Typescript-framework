## Mô tả (Description)

### Description:

As a user, when I need to view customer records, I want to see a list of all customers so that I can access and manage customer records.

### Context

This issue is for viewing the customer list in the Customer List & Management area.

### Acceptance criteria

1. A table displays the list of customers along with these attributes:

- Customer Name

- Primary Contact

- Phone

- Email

- Service Location (Primary Service Location)

- City, State

- Status (Active / Inactive)

- Action (Edit / Delete)

1. The user can sort any of the columns above.

2. Pagination is available on the list.

3. The user can navigate between pages.

4. A search bar is available with placeholder: “Search by Customer Name”.

5. Search filters the customer list based on Customer Name.

6. A Status filter is available. Options: Active / InactiveSelecting a status filters the customer list accordingly
